**[ PaPaGo ]**

아버지들의 재능 공유를 통해 다양한 체험의 기회를 제공하고 육아 참여를 유도하는 플랫폼 'PaPa Go'입니다. 
또한 쿠폰 제도를 도입한 결제 시스템으로 기업과 사용자에게 수수료 거래를 줄여 더 많은 수익 창출을 가능하게 하고 지속적인 클래스 참여를 유도하도록 구현했습니다.
